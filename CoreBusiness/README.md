# 📦 Core Business Logic

## 🎯 Objective

Apply business logic to optimize inventory management.

---

## 📊 ABC Analysis

Products are classified based on annual consumption value:

- A → High importance (top 80%)
- B → Medium importance
- C → Low importance

---

## 📦 Inventory Optimization

Implemented calculations:

- EOQ (Economic Order Quantity)
- Demand per day
- Safety stock
- Reorder point

---

## 🚦 Stock Status

Each product is classified as:

- OK → sufficient stock
- Surveillance → needs attention
- A commander → reorder required

---

This module transforms raw data into actionable insights for supply chain decision-making.
