# 🧼 Data Cleaning

## 🎯 Objective

Transform raw and inconsistent data into a clean and reliable dataset for analysis.

---

## ⚙️ Steps Performed

- Removed unnecessary columns
- Removed duplicate records
- Corrected negative stock values
- Replaced zero prices with null values
- Handled missing data:
  - Mode for categorical variables
  - Mean for numerical variables

---

## 📁 Files

- `data_cleaning_pd.py` → Python script (pandas)
- `stock_cleaned.csv` → Cleaned dataset

---

## 🧠 Outcome

The dataset becomes:

- Consistent
- Complete
- Ready for analysis

---

## 💡 Importance

Data cleaning is a critical step to ensure accurate and reliable results in any data analysis project.
