# 📊 Dataset

## 🎯 Description

This folder contains the raw dataset used for the inventory optimization project.

The dataset simulates a real-world inventory environment with:

- Multiple product categories
- Demand variability
- Pricing differences
- Supplier information

---

## ⚠️ Data Quality Issues

The dataset intentionally includes:

- Missing values
- Duplicate records
- Inconsistent values (negative stock, zero prices)

## 🎯 Purpose

This dataset is used to demonstrate:

- Data cleaning techniques
- Data preprocessing workflows
- Real-world data challenges in supply chain analytics
